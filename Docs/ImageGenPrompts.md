## Gameplay Spritesheet

```text
Use case: stylized-concept
Asset type: production-ready 1024x1024 transparent-ready pixel art spritesheet for an offline iOS action RPG asset catalog.
Primary request: Create a clean dark fantasy top-down pixel art spritesheet inspired by classic dungeon crawler mood without copying any existing franchise or copyrighted characters. The sheet must be easy to crop into game assets.
Canvas and grid: 1024x1024 square, crisp pixel art, nearest-neighbor aesthetic, 4 columns by 4 rows of equal 256x256 cells, thin neutral gray grid guides between cells, no text, no watermark.
Chroma-key background: Use a perfectly flat solid #00ff00 background in all empty areas for background removal. The background must be one uniform color with no shadows, gradients, texture, reflections, floor plane, or lighting variation. Keep subjects fully separated from background with crisp edges. Do not use #00ff00 anywhere in subjects or tiles.
Cells: hero knight, skeleton raider, ash warlock boss, archivist NPC, dungeon floor, cracked wall, iron gate, glowing shrine, health potion, mana potion, sword, ember ring, relic shard, slash effect, shadow projectile, boss arena floor.
Style constraints: low-resolution pixel art rendered at high resolution, strong silhouettes, limited palette with charcoal, moss green, iron gray, blood red, amber, teal highlights. No typography, no logos, no photorealism.
```

## UI Atlas

```text
Use case: stylized-concept
Asset type: production-ready 1024x1024 transparent-ready pixel art UI atlas for an offline iOS action RPG game asset catalog.
Primary request: Create a dark fantasy pixel-art HUD and interface atlas for a top-down dungeon crawler mobile game. The atlas must be easy to crop into UI textures.
Canvas and grid: 1024x1024 square, crisp pixel art, nearest-neighbor aesthetic, 4 columns by 4 rows of equal 256x256 cells, thin neutral gray grid guides between cells, no text, no letters, no numerals, no watermark.
Chroma-key background: Use a perfectly flat solid #00ff00 background in all empty areas and all frame interiors for background removal. The background must be one uniform color with no shadows, gradients, texture, reflections, floor plane, or lighting variation. Keep interface objects fully separated from background with crisp edges. Do not use #00ff00 anywhere in the UI objects.
Cells: HUD frame, health orb, mana orb, minimap frame, amber button, teal button, crimson button, disabled button, codex panel, quest tracker strip, loot popup frame, portrait frame, pause menu frame, inventory slot, selected inventory slot, mobile joystick ring.
Style constraints: low-resolution pixel art rendered at high resolution, strong readable silhouettes, cohesive with charcoal, iron gray, brass gold, blood red, mana blue, teal, amber. Avoid typography, logos, modern flat UI, photorealism, shadows outside the object silhouettes.
```
